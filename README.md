# AURA — Luxury Fine Dining & Restaurant Experience

A modern, highly polished, and visually stunning landing page for **AURA**, an upscale fine dining restaurant. This project showcases premium web design patterns, including sophisticated typography, a dark luxurious color palette, responsive bento grids, glassmorphism, and smooth scroll reveal animations.

## 🔗 Live Demo
[https://aura-luxury-fine-dining-restaurant.vercel.app/]

--

## ✨ Features

- **Luxury Aesthetics & Typography**: Designed with an elegant, dark aesthetic featuring gold accents, custom SVG iconography, and premium Google Fonts (`Inter` / `Playfair Display`).
- **Interactive Booking System**: A custom-validated reservation form featuring glassmorphism styling, real-time input checks, opening hours validation (17:00 – 22:30), and an animated success confirmation screen.
- **Bento Grid Gallery**: A modern, responsive showcase of restaurant ambiance, craftsmanship, wine pairings, and desserts.
- **Scroll Reveal Animations**: Smooth transitions and fade-ins powered by the browser's native **Intersection Observer API** for optimal performance.
- **Mobile First & Responsive**: Adaptable navbar with a hamburger menu transition that scales flawlessly from mobile displays up to ultra-wide desktops.
- **Contact & Map Integration**: Embedded interactive Google Maps location display along with detailed operational hours and reservations contact cards.
- **Newsletter Subscription**: Animated micro-interaction queue submission in the footer.

---

## 🛠️ Technology Stack

- **Markup**: Semantic HTML5 structures for SEO-friendly crawling and accessibility.
- **Styling**: Vanilla CSS3 using modern layouts (CSS Grid, Flexbox, Custom CSS Variables) with HSL color tokens.
- **Interactions**: Pure ES6+ JavaScript (no heavy third-party framework dependencies like jQuery or React required).
- **Fonts & Icons**: Custom inline SVG graphics and Google Fonts APIs.

## 🚀 Getting Started

### Prerequisites
You only need a modern web browser to run this project.

### Local Development
1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/aura-luxury-dining.git
   ```
2. **Navigate into the folder**:
   ```bash
   cd aura-luxury-dining
   ```
3. **Open the project**:
   - Simply double-click `index.html` to open it in your browser.
   - *Recommended:* Use the **VS Code Live Server** extension to launch a local development server for live updates.
